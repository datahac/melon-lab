## Getting Started

### Prerequisites

- Node.js

Get the code:

```bash
git clone git@github.com:melonproject/melon-lab.git
cp .env.defaults .env
```

### Setup

```bash
yarn
yarn dev
```

## Test

Run tests

```bash
yarn test
```
